# 🧠 CredExplain — Explainable AI Credit Scoring System

An IEEE-level machine learning pipeline for credit risk assessment 
using the German Credit Dataset, featuring SHAP/LIME explainability 
and cost-sensitive threshold optimization.

## 🚀 Features
- 5 ML models compared: Logistic Regression, Random Forest, 
  XGBoost, LightGBM, Stacking Ensemble
- Optuna hyperparameter tuning
- SHAP & LIME local and global explainability
- Cost-sensitive threshold optimizer
- AUC of 0.82+

## 🛠️ Tech Stack
- Python, Scikit-learn, XGBoost, LightGBM
- SHAP, LIME, Optuna
- Pandas, NumPy, Matplotlib
